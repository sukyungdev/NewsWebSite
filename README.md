# NewsWebSite

<img src="https://user-images.githubusercontent.com/96860670/175325036-f6cc3910-9361-4553-87e1-7ac9d50d458f.png" alt="img" width = "500px"/>

Implementation of news website using Web API.  
Vanilla Js와 news API를 이용하여 구현한 NewsWebSite입니다.

-   Using fetch to load API data and create website using HTML/CSS/JS
-   Create a responsive website made by bootstrap

Demo  
[https://sudev-news.netlify.app/](https://sudev-news.netlify.app/)

## skills

-   HTML
-   CSS
-   Bootstrap
-   JavaScript

## Major function

-   Showing the latest headline news (최신 뉴스 보여주기)
-   Showing news about the topic (토픽에 관한 뉴스 보여주기)
-   News Search Capability (뉴스 검색 기능)
-   Pagination
-   Responsive Web

## 배포

I used netlify to deploy.  
netlify를 이용하여 배포하였습니다.
